# CodeLive
Instructions For Running in Local Machine:
1. Clone the repository (or) Download the Project as zip folder on your Desktop
2. Unzip the Folder
3. Open Command Prompt
4. Type these Commands:
5. cd Desktop
6. cd CodeLive-main
7. cd CodeLive
8. npm install
9. npm start
